# Fluid-mechanics

В этом репозитории собраны две курсовые работы по теме решения уравнений гидродинамики.

Авторы:
- Александр Сенцов
- Илья Макаров

Кафедра: Вычислительная математика, МГУ.

Обе курсовые посвящены близким темам, поэтому они хранятся в одном репозитории.  
При этом реализации независимые: отличаются детали постановки, подход к имплементации и структура кода.
